## 项目概述

StyleVault 是一个 AI 驱动的时尚风格探索平台。用户上传照片即可获得 AI 风格分析、相似风格推荐和风格迁移效果预览。

## 技术架构

### 前端
- Vue 3 Composition API
- Pinia 状态管理
- Tailwind CSS 响应式设计

### 后端
- Python FastAPI 异步服务
- PyTorch 模型推理
- PostgreSQL + Redis 数据层

### AI 模型
- CLIP 图像编码
- 风格特征向量检索
- 轻量级风格迁移模型

## 关键特性

- 图片上传 + AI 风格分析
- 向量检索匹配相似风格
- 风格迁移效果预览
- 个性化风格推荐
- 收藏和历史记录
