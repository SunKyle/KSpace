## 项目概述

KChat 是一个实时聊天系统后端服务，使用 Java Spring Boot 构建。支持多房间管理、WebSocket 实时通信、消息持久化和在线状态追踪。

## 技术栈

- **Spring Boot 3** — 核心框架
- **WebSocket + STOMP** — 实时消息推送
- **PostgreSQL** — 消息和用户数据持久化
- **Redis** — 在线状态缓存和消息队列
- **Spring Security** — JWT 认证

## 核心功能

- 多聊天室创建和管理
- 实时消息收发（WebSocket）
- 用户在线状态显示
- 消息历史记录查询
- 文件/图片传输
- 未读消息计数
